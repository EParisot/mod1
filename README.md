# install
```
pip install -r requirements.txt
```

# usage
```
python3 mod1.py [map.mod1]
```
(empty map by default)

# Commands (start/stop)
F : Flood
R : Rain
W : Wave

# restrictions
Map points must be integers from [0, 99] (included)
